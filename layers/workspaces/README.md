# Workspaces Layer

This layer provides multi-tenant workspace management functionality.

## Features
- Workspace creation and management
- Team member invitation and role management
- Workspace switching with context persistence
- Integration with Auth and Entitlements

## Dependencies
- `@starter-nuxt-amplify-saas/auth`
- `@starter-nuxt-amplify-saas/entitlements`
- `@starter-nuxt-amplify-saas/uix`
